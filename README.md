Class schedul manager

Этот пакет предоставляет основные классы для работы с учебным расписанием в университете:

  * Event: описывает учебное занятие, позволяет устонавливать дату, время, предмет и место для подробного описания события
  * Schedule: управляет всем расписанием в целом, ищет нужную дисциплину, дабвляет или удаляет ее, а так же проверяет не занято ли уже указанное время
  * Teacher: управляет данными преподавателя, связывает преподавателя с конкретной парой, вывод  информацию о том какой преподаватель ведет какую дисциплину в расписании


Пакет также включает исполняемый файл `uni_schedul` для запуска демонстрации из командной строки.
